\# 💰 Loan Approval Prediction System using Machine Learning



\## 📌 Overview

This project predicts whether a loan application should be approved or rejected based on user input using a machine learning model integrated into a Django web application.

\## 🚀 Features

\## 🛠️ Technologies Used

\- Python  

\- Machine Learning  

\- Django

\- HTML/CSS  



\---



\## ⚙️ How it Works

\- User enters details (income, credit score, etc.)  

\- ML model processes input  

\- System predicts approval or rejection  



\---



\## ▶️ How to Run



\### 1. Install dependencies

pip install django pandas scikit-learn



\### 2. Apply migrations

python manage.py migrate



\### 3. Run server

python manage.py runserver



\### 4. Open in browser

http://127.0.0.1:8000/
